## 📝 **经验教训**

### **1. Gateway WebSocket 配对失败**
- **原因：** 客户端尝试升级权限时被网关拒绝
- **解决：** 执行 `openclaw gateway restart` 即可恢复
- **建议：** 如果频繁发生，检查网络连接和 Gateway 配置

### **2. LLM 模型超时问题**
- **原因：** 默认 60 秒超时阈值对某些复杂任务不够用
- **解决：** 将 `timeoutSeconds` 从 60s → 120s
- **建议：** 
  - 对于响应较慢的 API Provider，适当增加超时阈值
  - 监控日志观察是否仍有频繁超时
